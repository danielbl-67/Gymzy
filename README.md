# 🏋️‍♂️ Gymzy

**Gymzy** es una aplicación móvil para Android diseñada para ayudar a los usuarios a gestionar sus rutinas de entrenamiento, explorar ejercicios y realizar un seguimiento de su progreso físico de manera sencilla e intuitiva.

---

## 🚀 Características

- **Gestión de Ejercicios:** Visualiza y organiza ejercicios basados en diferentes grupos musculares.
- **Interfaz Intuitiva:** Diseño limpio enfocado en la experiencia del usuario durante el entrenamiento.
- **Base de Datos en Nube:**Firebase Almacenamiento de rutinas y progresos.
- **Guía Visual:** Incluye recursos visuales para la correcta ejecución de los ejercicios.

## 🛠️ Tecnologías Utilizadas

- **Lenguaje:** [Java]
- **Entorno de Desarrollo:** Android Studio
- **Gestor de Dependencias:** Gradle (Kotlin DSL)
- **Recursos:** Biblioteca de imágenes de anatomía muscular.

## 📦 Estructura del Proyecto

El proyecto sigue la estructura estándar de una aplicación Android:

- `/app`: Contiene el código fuente de la aplicación (Java), layouts (XML) y recursos.
- `/gradle`: Configuración del motor de construcción.
- `imgGYMZYMusculos.rar`: Paquete de recursos gráficos utilizados para identificar grupos musculares.
